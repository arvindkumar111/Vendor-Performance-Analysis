# Vendor-Performance-Analysis

Project Overview
Vendor Analysis is a comprehensive data analysis project focusing on evaluating vendor pricing strategies and various performance metrics to support strategic decision-making. The project leverages SQL for data extraction and transformation, Python for exploratory data analysis and statistical insights, and Power BI for interactive visualization and reporting.

Key Objectives
Analyze vendor pricing strategies to identify competitive and cost-effective pricing models.

Examine patterns in vendor purchase quantities, costs, and sales performance.

Identify top-performing and low-performing vendors based on pricing and sales data.

Uncover insights on pricing impacts on inventory costs and profitability.

Support business decisions through data-driven vendor management and pricing optimization.

Technologies Used
SQL: Efficient querying and transformation of raw vendor and sales data from relational databases.

Python: Data cleaning, exploratory data analysis (EDA), hypothesis testing, and automation of analytical workflows.

Power BI: Creation of dynamic dashboards and reports to visualize vendor performance metrics and pricing strategies.
